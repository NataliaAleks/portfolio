\## 📁 Проекты



\### 1. Анализ рынка вин ЮАР (SQL)



\*\*Задача:\*\* выбрать сорта винограда для посадки на новой винодельне в ЮАР.

\*\*Что сделано:\*\*

\- Сегментация рынка по цене и качеству (Premium, Value, Economy, Overpriced)

\- Расчёт отклонения от диагонали «цена / качество»

\- Отбор сортов по метрикам: цена, качество, стабильность, доля рынка

\- Рекомендации по формированию портфеля



\*\*Навыки:\*\* SQL (оконные функции, CTE, регрессия), бизнес-логика, работа с рыночными данными

\### 2. Анализ рынка вин: Python EDA



\*\*Задача:\*\* исследовать структуру рынка вин, сравнить Старый и Новый свет, проверить гипотезы.

\*\*Что сделано:\*\*

\- Предобработка данных, создание сегментации (age)

\- EDA с графиками (цены, оценки, сорта)

\- Проверка гипотез (t-test, нормальность, Левен)

\- Корреляция цена/качество



\*\*Навыки:\*\* Python (pandas, matplotlib, seaborn, scipy.stats), EDA, визуализация, статистика
